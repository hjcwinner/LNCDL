## 주요기능
----------
* Login 기능
* Navigator를 통한 페이지 이동
* Count App을 통한 숫자 증가 감소
* Dice App를 통한 주사위 게임
* Lotto App를 통한 회차별 번호(Api) 불러오기 및 추첨번호 생성하기
